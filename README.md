# Vinícola Serra Dourada

Landing page de uma vinícola fictícia, com catálogo de produtos, filtros, carrinho lateral e agendamento.

## Funcionalidades

- Catálogo de vinhos
- Filtros e ordenação
- Carrinho lateral
- Persistência com `localStorage`
- Modal de agendamento
- Layout responsivo

## Tecnologias

- React
- JavaScript
- Vite
- Tailwind CSS

## Executando localmente

```bash
git clone https://github.com/WessYu/vinicola-serra-dourada-main.git
cd vinicola-serra-dourada-main
npm install
npm run install:app
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run api
```
